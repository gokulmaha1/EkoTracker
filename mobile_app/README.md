## Building and Installing the APK

The APK file for Android can be found at:
`build/app/outputs/flutter-apk/app-release.apk`

To install it on your device:
1. Transfer the APK file to your Android phone.
2. Open the file using a file manager.
3. Allow installation from unknown sources if prompted.
4. The app will be installed.

**Note:** Ensure you have the backend server running and accessible from the mobile device (e.g., via IP address or hosted URL).
